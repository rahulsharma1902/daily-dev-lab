# Notes & Documentation

This directory contains learning notes, documentation, and knowledge base entries.

## Structure

```
notes/
├── til/           # Today I Learned entries
│   └── YYYY-MM/   # Organized by month
└── topics/        # Deep-dive topic notes (optional)
```

## TIL (Today I Learned)

Short daily entries documenting something new learned each day.

### Quick Start

Create a new TIL entry:
```bash
python scripts/daily_commit.py --type til --topic "topic name"
```

### Format

Each TIL includes:
- Date and category
- What was learned
- Key takeaways
- Code examples (if applicable)
- Resources

## Benefits

- Reinforces learning through writing
- Creates searchable knowledge base
- Tracks learning progress over time
