# Experiments

This directory contains explorations of new technologies, libraries, and concepts.

## Categories

### Python
- New library explorations
- Framework experiments
- Language feature deep-dives

### JavaScript
- Framework experiments
- Node.js explorations
- Frontend concepts

## What Belongs Here

- Technology spikes
- Proof of concepts
- Library evaluations
- Learning experiments

## File Naming

```
YYYYMMDD_topic_name.py
```

Example: `20240115_fastapi_basics.py`

## Quick Start

Create a new experiment:
```bash
python scripts/daily_commit.py --type experiment --topic "topic name"
```
