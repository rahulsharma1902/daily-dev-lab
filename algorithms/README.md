# Algorithms

This directory contains implementations of classic algorithms, organized by category.

## Categories

### Sorting
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort

### Searching
- Linear Search
- Binary Search
- Jump Search
- Interpolation Search

### Dynamic Programming
- Fibonacci
- Coin Change
- Longest Common Subsequence
- Knapsack Problem

## Adding New Algorithms

When adding a new algorithm:
1. Create a file with a descriptive name
2. Include complexity analysis in comments
3. Add test cases
4. Document the approach

## File Naming Convention

```
algorithm_name.py
```

Example: `merge_sort.py`, `binary_search.py`
