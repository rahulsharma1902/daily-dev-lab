# Code Snippets

A personal library of reusable code patterns and utilities.

## Categories

### Python
- API helpers
- File operations
- Data processing
- Utility functions

### JavaScript
- DOM utilities
- Async patterns
- API helpers

## Usage

Import snippets as needed:
```python
from snippets.python.api_helper import make_request
```

## Quick Start

Create a new snippet:
```bash
python scripts/daily_commit.py --type snippet --topic "snippet-name" --lang python
```

## Best Practices

1. Keep snippets focused and single-purpose
2. Include docstrings and examples
3. Add type hints (Python) or JSDoc (JavaScript)
4. Test before adding
