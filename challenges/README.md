# Coding Challenges

This directory contains solutions to coding challenges from various platforms.

## Organization

Challenges are organized by month:
```
challenges/
├── 2024-01/
│   ├── 01_two-sum.py
│   ├── 02_reverse-linked-list.py
│   └── ...
├── 2024-02/
└── ...
```

## Platforms

- LeetCode
- HackerRank
- CodeSignal
- Codewars
- Project Euler

## Solution Template

Each solution includes:
- Problem description
- Approach explanation
- Time/Space complexity
- Test cases

## Quick Start

Create a new challenge entry:
```bash
python scripts/daily_commit.py --type challenge --name "problem-name"
```
